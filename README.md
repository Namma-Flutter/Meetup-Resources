## About

[Namma Flutter](https://www.nammaflutter.in) is a Google-recognized Flutter developer community rooted in Chennai, Tamil Nadu. We unite developers, designers, and enthusiasts through meetups, hackathons, and workshops across South India.

This repository is the central archive of resources — slides, code samples, links, and notes — shared by speakers and contributors at our events. Every resource lives here so no knowledge gets lost after an event ends.

**Community at a glance**

- 5,000+ members
- 70+ speakers
- 20+ workshops and meetups

## Repository Structure

```
event_resources/
└── {year}/
    └── {month}/
        └── {event-slug}/
            ├── README.md                        # Event overview, session list & speaker links
            └── {session-title_speaker-name}/    # Slides, code, or a README with resource links
```

Each event folder contains a `README.md` that lists all sessions and speakers. Resources for each session go into a subfolder or as a direct file, named after the session and speaker.
