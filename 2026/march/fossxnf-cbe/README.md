# FOSS x Namma Flutter — Coimbatore

**Kaiya Kudunga, Sagala**

**Date:** 28th March 2026

**Location:** [iamneo](https://maps.app.goo.gl/W9vgpNsQ3h4WdfPx7),Coimbatore, Tamil Nadu

**Type:** Collaborative Event
